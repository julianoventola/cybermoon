# Cybermoon Theme

A High dark contrast theme for Visual Studio Code.

![theme preview](https://i.imgur.com/0vz6SoT.png)

</div>

# Installation

- Click the `Install` button
- Use `Ctrl + Shift + P`, type "Preferences: Color Theme" and select **Cybermoon**
